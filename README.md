# 🛡️ Strategic Shield

**India's Critical Infrastructure Protection & Personal Wealth Intelligence Platform**

Strategic Shield is a high-fidelity, cinematic AI intelligence platform designed to map, simulate, and analyze threats to India's most critical economic sectors. From state-level dependency networks to individual portfolio risk, SENTINEL AI provides actionable strategic insight.

## 🚀 Key Features

### 1. The Gateway & Identity Verification
- **Cinematic Entry:** Triple-stage animated sequence leading from global secure uplink to the briefing terminal.
- **Secure Identity Gate:** JWT-based authentication for strategic advisors, with a "Bypass" mode for anonymous guest reconnaissance.

### 2. Dependency Network Graph
- **Real-time Mapping:** Visualization of 50+ critical listed entities (Reliance, HDFC, BDL, etc.) and their cross-sector dependencies.
- **Vulnerability Scoring:** Automated risk profiling based on fiscal, operational, and geopolitical markers.

### 3. Tactical Scenario Engine
- **AI Simulation:** Generate high-impact threat scenarios (kinetic, cyber, or supply chain) for specific corporations.
- **Downstream Impact analysis:** Predict cascading economic failures across the Indian ecosystem.

### 4. Personal AI Advisor (NEW)
- **Asset Registration:** Private tracking of personal stock holdings (NSE).
- **Correlation Scan:** Specialized LLM analysis comparing your portfolio markers against geopolitical threat vectors.
- **Wealth Protection:** Actionable intelligence derived from live price action and threat signals.

## 🛠️ Setup & Security

### 1. Environment
Create a `.env` file from the provided template:
```bash
SECRET_KEY=your_secure_hex
GEMINI_API_KEY=your_google_ai_key
GROQ_API_KEY=your_groq_ent_key
```

### 2. Installation
```bash
pip install -r requirements.txt
python -m backend.app
```

### 3. Deployment Notes
- **Database:** Local SQLite (`strategic_shield.db`) is used for rapid tactical scaling.
- **Security:** Hashed passwords via Passlib (Bcrypt) and JWT sessions.
